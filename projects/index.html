Ask IPL Stats - A question answering system for IPL data

Built an end-to-end text to SQL pipeline for IPL statistics data using LLMs. Ask any question about IPL statistics in english and get the relevant data.
  • Developed ETL pipelines to get ball by ball IPL data from cricsheet, cricinfo API.
  • Implemented a classifier model to filter out unrelated questions before passing to LLM.
  • Built a natural language to sql pipeline using Gemini LLMs to convert the relevant questions into sql queries and fetch required data.
  • Optimized the pipeline using caching to reduce repeated calls to LLM for similar questions.
  • Enhanced the pipeline by adding NER tagging to extract the player/team names from the questions to enable fetching data for wrongly spelt names.
